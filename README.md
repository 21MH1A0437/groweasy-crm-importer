# groweasy-crm-importer
AI-powered CRM Importer for uploading, mapping, reviewing, and importing customer data.
##  About the Project

GrowEasy AI CRM Importer is a web application that helps users upload and import customer data into a CRM system. It supports CSV, XLS, and XLSX files and provides a simple workflow for previewing, reviewing, and importing records.

##  Live Demo

https://groweasy-crm-importer-248945226943.asia-southeast1.run.app/

##  Features

- Upload CSV, XLS, and XLSX files
- Preview uploaded data
- AI-powered data processing
- Review records before import
- Responsive user interface

##  Technologies Used

- React.js
- Next.js
- TypeScript
- Tailwind CSS
- Google Gemini AI
- Papa Parse
- SheetJS (XLSX)
